Aqui veremos os tipos básicos de variáveis e o que elas significam

Não adianta muito olhar no navegador, mas ok!

VAR é de variável, na época das cavernas o JS usava assim, as vezes tu ainda vê, mas variáveis criadas por VAR são muito amplas e tem chance de dar problema

CONST é de constante, são usadas para variáveis que não irão mudar, e é feito para não ser alterado, então se tentarem mudar tem que reclamar

LET é de sei lá... vou procurar aqui... Foda-se, uma galera de matemática pegou isto do SCALA e foi, é tipo o CONST, mas podendo alterar

Qual a diferença VAR vs LET vs CONST? Var é amplo demais, então serve em todo o arquivo em que estiver. Const e Let é preso ao ESCOPO

O que é ESCOPO? É um pedaço delimitado do código por chaves { }, if usa, classe usa, função usa e por ai vai, tem { }, é um ESCOPO
