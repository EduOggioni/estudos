Todas as tags do html, para serem manipuladas, precisam de alguma forma de identificação.

Ids são usados para identificação única, logo duas tags HTML não podem ter o mesmo ID.

Já classes são para determinar algo que pode ser usado para identificação de múltiplas tags HTML.
